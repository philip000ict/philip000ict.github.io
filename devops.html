Design for Devops Thin Egress

Using minimal bandwidth to maintain effective system controln in response to limited technical capability has long been a core performance requirement, from remote location in Australia to a research project in the Northern jungles of Laos (2015). Predominantly a key component of this deployment activity is the use of CLI over SSH protocol. This is superior to the use of a proprietary GUI or portal because the data transmission volume of CLI commands are in the range of Kilobits and less, whereas the use of a browser GUI portal requires megabytes of bandwidth and network speed for an acceptable user experience
